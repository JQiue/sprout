# Agent

## API

| 路由                    | 作用                                                                                                              |
| ----------------------- | ----------------------------------------------------------------------------------------------------------------- |
| `GET /api/heartbeat`    | 返回 Agent 的状态                                                                                                 |
| `POWT /api/upload/init` | 为某个站点初始化上传 token，包含 site_id                                                                          |
| `POST /api/upload/file` | 为某个站点上传文件，流程：验证 token，保存上传文件，更新部署状态，审核文件内容，申请测试域名，创建 Nginx 配置文件 |
